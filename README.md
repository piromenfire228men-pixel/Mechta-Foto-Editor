# Mechta-Foto-Editor
Официальный репозитор программы Mechta Foto Editor. фоторедактор любительского уровня, написанный на языке программирования Python, предназначенный для работы на слабых компьютерах. Включает систему резервного копирования файлов, систему регенерации файлов и консоль восстановления. Работает нативно на операционных системах Windows и Linux (поддерживается только 64-битная архитектура).

Система резервного копирования файлов:

Она сохраняет все файлы, которые вы когда-либо добавляли в проект. При удалении файлов с устройства, фотографии сохраняются в специальной папке MF_Rezerv, откуда пользователь не сможет удалить или испортить файлы.

Система регенерации файлов:

Позволяет забыть о таких ошибках, как «Критическая ошибка: не найден файл „имя.json“». Эта система автоматически восстанавливает все .json-файлы, файлы кеша и файлы из папки MF_Rezerv при нажатии сочетания клавиш Ctrl+Shift+O.

Консоль восстановления:

Представляет собой систему для ручного восстановления всех файлов программы в удобном для пользователя порядке. С её помощью можно запустить «Систему регенерации файлов». Консоль не имеет доступа к командам и файлам пользователя и операционной системы, работает только в пределах файлов программы.

Mechta Foto Editor Open не планируется портировать на мобильные платформы, такие как телефоны и планшеты на Android или iOS.
